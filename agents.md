# Agent rules

## Communication
- Be concise. Sacrifice readability and vocabulary for brevity.
- No preamble, no summaries, no affirmations. Get to the point.

## Planning
- Plans must be short bullet lists. Cut every non-essential word.
- Prefer one concrete sentence over a paragraph of intent.

## Testing
- Every new feature or bug fix gets tests in `tests/`.
- Run `pytest` before declaring a task done. It must be green.
- Test command: `.venv/bin/python -m pytest`
